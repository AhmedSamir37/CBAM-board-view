CBAM Board View (Steel - EAF/DRI)

Decision-grade CBAM exposure calculator for steel exports to the EU.

1) What is this app?

CBAM Board View is a lightweight dashboard designed to estimate the potential CBAM cost exposure for steel exports to the European Union.

It is intentionally simplified for board-level stress-testing and decision support.

4) Scenarios

The app provides three scenario modes:

  • Base: Current input values.
  • Conservative: Lower carbon price / better performance assumptions.
  • Stress: Higher carbon price / worse performance assumptions.

5) Actions

  • Edit inputs: Show/hide the inputs panel.
  • Apply: Calculate KPIs and scenarios.
  • Reset: Clears all inputs and results.
  • Copy share link: Copies a URL containing saved scenario data.
  • Download CSV: Export inputs and KPIs into a CSV file.
  • Print / Save PDF: Uses browser printing to save the report as PDF.
